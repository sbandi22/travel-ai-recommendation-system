# Travel Outfit Engine

An AI-powered travel outfit recommendation system that predicts outfit categories
from trip context and generates photorealistic fashion images for both men and women.

## How It Works

```
Itinerary event (temp, weather, activity, time)
        │
        ▼
 XGBoost classifier  ──▶  outfit category (Casual / Formal / Athletic / Rain Ready)
        │
        ▼
 Prompt builder      ──▶  structured fashion prompt (gender-aware, style variant)
        │
        ▼
 HF Inference API    ──▶  generated outfit image (FLUX.1-schnell)
        │
        ▼
 Packing optimizer   ──▶  deduplicated packing list across all events
```


## Setup

```bash
# 1. Create and activate virtual environment
python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # Mac/Linux

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set your Hugging Face API key
set HF_API_KEY=hf_...          # Windows cmd
# export HF_API_KEY=hf_...     # Mac/Linux
```

## Run

```bash
python test_engine.py
```

Generates male and female outfit images for each event in the itinerary
and prints a final packing list.

## Regenerate Model (optional)

```bash
python data/synthetic_generator.py   # recreates data/raw/outfit_data.csv
python models/train.py               # retrains and saves model
```

Wait — `synthetic_generator.py` was moved:

```bash
python scripts/generate_training_data.py   # recreates data/raw/outfit_data.csv
python models/train.py                     # retrains and saves model
```

## Preview Prompts (no API key needed)

```bash
python scripts/show_example_prompts.py
```

## Style Variants

Each outfit category has 5 named style variants. Running with `n=2` or more
generates distinct looks per event:

| Category   | Variants |
|------------|----------|
| Casual     | Streetwear, Minimalist, Smart Casual, Earth Tones, Monochrome |
| Formal     | Classic Business, Elegant Evening, Modern Professional, Power Suit, Creative Professional |
| Athletic   | Performance Run, Gym Training, Outdoor Active, Athleisure, Court Sport |
| Rain Ready | Sleek Urban Rain, Colourful Wet Weather, Trench Classic, Layered Storm Proof, Minimalist Rain |

## Model

- **Algorithm**: XGBoost classifier wrapped in a scikit-learn Pipeline
- **Features**: temperature, rain probability, activity type, weather condition, time of day
- **Labels**: Casual, Formal, Athletic, Rain Ready
- **Training data**: 10,000 synthetic samples generated by rule-based logic
